# shide

[![License](https://img.shields.io/badge/license-ISC-blue)](https://opensource.org/licenses/ISC)

This template is intended for use with the hase static site generator.

## Overview

shide provides clean, readable typography and layout for documents and small sites.

## Features

- Light and dark color modes
- Responsive container and flexible grid system
- Headings, lists, tables, code blocks, blockquotes, and forms styled
- Skip links and focus-visible outlines for accessibility
- Media queries for reduced motion and narrow viewports
- Small, dependency-free, and easy to integrate
